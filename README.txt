
📦 HEDEF TESPİTİ SİMÜLASYONU PROJESİ (İHALE DEMOSU)

Bu proje, uydu benzeri görüntülerde hedef (araç vs.) tespiti yapar ve çıktıların koordinat, sınıf ve güven skorlarını raporlar.

📌 Kurulum:
pip install ultralytics pillow

🚀 Çalıştırmak için:
python detect.py

📁 Çıktılar:
- /sonuc klasöründe tespit edilmiş görseller
- rapor.csv: Her hedefin görüntü adı, sınıfı, koordinatları ve skoru
